LLM vs ChatModel 区别

📝
LLM
Large Language Model
传统的文本进-文本出模型，输入一段文字，输出一个完整的文本回复。
  • 输入：纯文本字符串
  • 输出：纯文本字符串
  • 适合：文本生成、摘要、翻译
  • 示例：text-davinci-003
💬
ChatModel
Chat Model / 对话模型
基于消息的对话模型，更适合构建聊天机器人，支持多轮对话和上下文管理。
  • 输入：消息列表 [System, User, AI]
  • 输出：AIMessage 对象
  • 适合：聊天机器人、智能助手
  • 示例：gpt-4o, qwen3-max

🎯 典型使用场景

🔍 RAG 问答系统
🤖 客服机器人
📧 邮件撰写助手
💻 代码生成助手
