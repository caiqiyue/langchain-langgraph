LangChain 1.0 生态概览

🧠
模型层 Models
统一模型抽象层
为所有模型提供标准化调用，覆盖文本、多模态、Embedding 等多类型模型
  • ✅ 统一抽象：init_chat_model() 适配20+模型厂商
  • ✅ 异步/流式/批处理：ainvoke()，stream(), batch()
  • ✅ 扩展能力：with_structured_output()、Tool Calling
🔧
工具层 Tools
可执行 Agent 环境的关键能力层
工具系统提供统一 Tool 抽象，支持所有主流模型的 Tool Calling
  • ✅ 内置工具：搜索、计算、代码执行等100+工具
  • ✅ 自定义工具：@tool装饰器 / BaseTool / ToolNode
  • ✅ 工具包：Toolkit（如GitHub、Slack集成）
💾
记忆层 Memory
统一 State 管理
提供对话记录、长期检索、多模态 Memory 等能力，支持持久化
  • ✅ 短期记忆：消息历史自动管理
  • ✅ 长期记忆：向量数据库存储（Chroma, Pinecone）
  • ✅ 存储接口：Store（跨会话持久化）
🤖
Agent层 Agents
create_agent 核心接口
以create_agent为核心接口，基于LangGraph构建统一Agent抽象
  • ✅ 核心API：create_agent()
  • ✅ 执行引擎：LangGraph Runtime
  • ✅ 中间件：Middleware（HITL、压缩、路由）
🔄
工作流层 Workflows
图结构编排
以 StateGraph 为核心画布，将业务逻辑解耦为"节点+边+状态"
  • ✅ 简单链：Chain（快速串联）
  • ✅ 复杂图：LangGraph（条件分支、循环）
  • ✅ 模板库：LangChain Hub（共享Agent模板）
🔍
调试监控层 Debugging
全链路可观测性
深度集成 LangSmith 平台，自动捕获链与图的每一步状态
  • ✅ 本地日志：verbose=True
  • ✅ 云端平台：LangSmith（可视化链路追踪）
  • ✅ 评估工具：LangChain Evaluate

💡 核心价值

让开发者用10行代码完成原本需要1000行代码的AI应用，并且自动获得状态持久化、人工干预、并发控制等企业级能力。
