⚡ astream_events() 语义事件流

异步流式监听完整语义生命周期事件

  • 🔗
    on_chain_start
    链开始执行
  • 📝
    on_prompt_start / on_prompt_end
    Prompt 处理开始/结束
  • 🤖
    on_llm_start
    LLM 调用开始
  • 📊
    on_llm_stream（逐 Token）
    每个 token 生成时触发
  • ✅
    on_llm_end
    LLM 调用完成

🎯 适用场景

  • • 调试 LLM 推理过程
  • • 了解 LangChain pipeline 的执行顺序
  • • 构建 UI（如 web 前端的逐 token streaming）
  • • 实现日志、可观测性、监控系统
