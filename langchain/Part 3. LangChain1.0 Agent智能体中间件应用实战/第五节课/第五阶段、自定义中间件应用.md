## <center>第五阶段、自定义中间件应用

### 1. ModelRequest 与 ModelResponse 参数传递

#### 1.1 数据结构详解

| 类型 | 作用域 | 生命周期 | 可修改性 |
|------|--------|----------|----------|
| ModelRequest | 单次模型调用 | 瞬态 | 可直接修改 |
| ModelResponse | 单次模型调用 | 瞬态 | 只读 |
| AgentState | 整个Agent会话 | 持久化 | 通过返回值更新 |

**ModelRequest 结构**：

```python
request.model      # 当前要调用的模型实例（可修改）
request.messages   # 消息列表（可修改）
request.tools      # 可用工具列表（可修改）
request.state      # Agent 的当前状态（只读）
request.runtime    # 运行时上下文
```

**AgentState 结构**：

```python
{
    'messages': [...],           # 完整对话历史
    'user_info': {...},          # 用户信息（由中间件注入）
    'user_permissions': [...],   # 用户权限（由中间件注入）
    'checkpoint': 'xxx'          # 用于恢复
}
```

#### 1.2 request.override() 方法

```python
# 修改模型
request = request.override(model=new_model)

# 修改消息
request = request.override(messages=new_messages)

# 修改工具
request = request.override(tools=new_tools)

# 链式调用
request = request.override(
    model=new_model,
    messages=new_messages
)
```

### 2. @dynamic_prompt 装饰器

#### 2.1 动态提示词中间件

@dynamic_prompt 装饰器自动处理 ModelRequest/Response 的封装，返回字符串作为动态 System Prompt。

```python
@dynamic_prompt
def role_based_prompt(request: ModelRequest):
    user_role = request.runtime.context.get("user_role", "user")

    prompts = {
        "expert": "你是一个专业气象分析师...",
        "beginner": "你是一个友善的天气助手...",
        "default": "你是一个简洁的天气助手..."
    }
    return prompts.get(user_role, prompts["default"])
```

#### 2.2 @dynamic_prompt vs @wrap_model_call

| 维度 | @dynamic_prompt | @wrap_model_call |
|------|-----------------|------------------|
| 返回值 | 字符串（System Prompt） | ModelResponse 对象 |
| 复杂度 | 简单 | 复杂 |
| 适用场景 | 动态提示词 | 完整中间件逻辑 |
| 可修改内容 | 仅 System Prompt | 模型、消息、工具等 |
| 灵活性 | 有限 | 完全灵活 |

### 3. 模型动态切换中间件

#### 3.1 @wrap_model_call 实现

```python
@wrap_model_call
def dynamic_model_router(request, handler):
    # 获取当前对话状态
    state = request.state
    messages = state.get("messages", [])

    # 切换条件判断
    should_use_large = False

    # 条件1: 长对话（超过5轮）
    if len(messages) > 5:
        should_use_large = True

    # 条件2: 包含复杂任务关键词
    elif messages:
        last_msg = messages[-1].content if messages else ""
        hard_keywords = ("证明", "推导", "严谨", "规划", "复杂")
        if any(kw in last_msg for kw in hard_keywords):
            should_use_large = True

    # 执行模型切换
    if should_use_large:
        request = request.override(model=large_model)

    return handler(request)
```

#### 3.2 切换效果对比

| 场景 | 模型选择 | Token 成本 | 响应速度 |
|------|---------|-----------|---------|
| 简单问答 | 小模型 | $0.001 | 快 |
| 包含"证明"的数学推导 | 大模型 | $0.01 | 慢 |
| 5轮以上对话 | 大模型 | $0.01 | 慢 |
| 日常闲聊 | 小模型 | $0.001 | 快 |

### 4. persist_session 会话持久化

#### 4.1 PersistSessionMiddleware 实现

```python
class PersistSessionMiddleware(AgentMiddleware):
    def __init__(self, path: str):
        super().__init__()
        self.path = path
        os.makedirs(path, exist_ok=True)

    def after_model(self, state: AgentState, runtime) -> None:
        timestamp = int(time.time())
        messages = state.get("messages", [])
        if not messages:
            return

        serialized_msgs = []
        for msg in messages:
            msg_data = {"role": msg.type, "content": msg.content}
            serialized_msgs.append(msg_data)

        filename = os.path.join(self.path, f"state_{timestamp}.json")
        with open(filename, "w", encoding="utf-8") as f:
            json.dump(serialized_msgs, f, ensure_ascii=False, indent=2)

def persist_session(path: str = "./sessions"):
    return PersistSessionMiddleware(path)
```

### 5. IT运维Agent综合案例

#### 5.1 中间件组合

```python
middlewares = [
    # === before_agent ===
    SecurityGuardrail(),           # 1. 安全检查
    RBACMiddleware(),             # 2. 权限验证

    # === before_model ===
    RBACMiddleware(),             # 3. 注入用户信息

    # === wrap_model_call ===
    dynamic_system_prompt,        # 4. 动态提示词
    dynamic_model_router,         # 5. 智能模型切换
    custom_context_middleware,    # 6. 上下文管理

    # === after_model ===
    ResponseValidator(),          # 7. 响应验证
    AuditLogger(),                # 8. 审计日志
]
```

#### 5.2 各中间件职责

| 序号 | 中间件 | Hook点 | 职责 |
|-----|-------|--------|------|
| 1 | SecurityGuardrail | before_agent | 危险操作关键词检测 |
| 2 | RBACMiddleware | before_agent | 用户权限验证 |
| 3 | RBACMiddleware | before_model | 注入 user_info |
| 4 | dynamic_prompt | wrap_model_call | 角色专属提示词 |
| 5 | dynamic_model_router | wrap_model_call | 长对话切换大模型 |
| 6 | ContextEditing | wrap_model_call | 清理旧工具结果 |
| 7 | ResponseValidator | after_model | 工具调用权限验证 |
| 8 | AuditLogger | after_model | 操作审计日志 |
