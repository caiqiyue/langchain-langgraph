## <center>第四阶段、中间件的工程化使用

### 1. before_model 中间件

#### 1.1 SummarizationMiddleware - 上下文压缩

`SummarizationMiddleware` 是 `before_model` 中间件的典型代表，在模型调用前对对话历史进行压缩。

**核心功能**：
- 自动检测消息 token 数量
- 超过阈值时触发压缩机制
- 保留最近 N 条关键消息
- 将旧消息压缩为摘要

**配置参数**：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| messages_to_keep | 5 | 保留最近消息数 |
| max_tokens_before | 2000 | 触发压缩的 token 阈值 |
| summary_prompt | 预设模板 | 摘要生成提示词 |
| model | 主模型 | 用于摘要的模型（可选） |

```python
summarization_middleware = SummarizationMiddleware(
    messages_to_keep=5,
    summary_prompt="请将以下对话历史进行摘要，保留关键决策点：\n\n{messages}\n\n摘要:"
)
```

#### 1.2 PIIMiddleware - 数据脱敏

PII（Personally Identifiable Information）中间件在数据离开系统前进行脱敏处理。

**处理内容**：
- 身份证号
- 手机号码
- 邮箱地址
- 银行卡号

#### 1.3 ModelCallLimitMiddleware - 模型调用限制

防止 Agent 进入死循环或过度消耗 token。

**限制策略**：
- 最大调用次数限制
- 时间窗口内的调用频率限制
- Token 预算上限

### 2. wrap_model_call 中间件

#### 2.1 ContextEditingMiddleware - 上下文管理

在模型调用过程中动态编辑上下文，管理对话状态。

**核心能力**：
- 实时修改上下文窗口
- 插入系统级指令
- 清理敏感信息

#### 2.2 ModelFallbackMiddleware - 故障切换

当主模型不可用时自动切换到备用模型。

```python
ModelFallbackMiddleware(
    primary_model="deepseek-chat",
    fallback_model="claude-3-haiku",
    timeout=30
)
```

#### 2.3 LLMToolSelectorMiddleware - 智能工具选择

根据用户意图动态选择最合适的工具子集。

**决策流程**：
1. 分析用户 Query
2. 评估工具与 Query 的匹配度
3. 选择 Top-N 最相关工具
4. 注入到 Agent 工具列表

### 3. wrap_tool_call 中间件

#### 3.1 ToolRetryMiddleware - 自动重试

工具调用失败时自动重试，提高执行成功率。

```python
ToolRetryMiddleware(
    max_retries=3,
    retry_delay=1.0,
    exponential_backoff=True
)
```

**重试策略**：
- 立即重试（网络瞬断）
- 指数退避（服务繁忙）
- 最大次数限制

#### 3.2 LLMToolEmulator - 工具模拟

在正式调用前模拟工具执行，验证参数正确性。

### 4. after_model 中间件

#### 4.1 HumanInTheLoopMiddleware - 人工审批

高风险操作触发人工审批流程。

**触发条件**：
- 删除操作
- 大额交易
- 权限变更
- 批量操作

#### 4.2 ToolCallLimitMiddleware - 工具调用限制

限制特定工具的调用频率和总量。

### 5. 中间件配置组合

```python
from langchain.agents import create_agent
from langchain.agents.middleware import (
    SummarizationMiddleware,
    PIIMiddleware,
    ToolRetryMiddleware,
    HumanInTheLoopMiddleware
)

agent = create_agent(
    model=model,
    tools=tools,
    middleware=[
        SummarizationMiddleware(messages_to_keep=5),   # 上下文压缩
        PIIMiddleware(),                               # 数据脱敏
        ToolRetryMiddleware(max_retries=3),           # 自动重试
        HumanInTheLoopMiddleware(),                   # 人工审批
    ]
)
```

**配置要点**：
- `before_model` 中间件按配置顺序正序执行
- `wrap_*` 中间件嵌套执行
- `after_model` 中间件逆序执行
