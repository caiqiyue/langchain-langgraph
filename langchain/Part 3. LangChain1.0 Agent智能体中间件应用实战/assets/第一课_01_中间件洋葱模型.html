📋 Hook 点详解

Hook 点作用常见中间件
before_agentAgent 执行前最后机会修改输入安全检查、权限验证
before_model模型调用前修改 prompt / 选择模型SummarizationMiddleware、PIIMiddleware
wrap_model_call包裹模型调用（异常处理、重试）ModelFallbackMiddleware
wrap_tool_call包裹工具调用（异常处理、重试）ToolRetryMiddleware
after_model模型调用后修改输出HumanInTheLoopMiddleware
after_agentAgent 执行完成后的最终处理日志记录、审计追踪
