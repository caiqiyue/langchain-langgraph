AGENTIC RAG REACT LOOP

Thought → Action → Observation → Done?

🧠 LLM 决策 THOUGHT
🔍 执行检索 ACTION
📊 评估结果 OBSERVE
❓ 判断结束 DONE?
🔄 AGENT
// STEP 01

🧠 LLM 决策 (Thought)

Agent 分析用户问题，理解意图，决定是否需要检索以及使用什么工具

model.bind_tools([rag_tool])
decision = agent.invoke()
// STEP 02

🔍 执行检索 (Action)

根据决策调用相应工具执行检索，从向量数据库获取相关文档

retriever.get_relevant
_documents(query)
// STEP 03

📊 观察评估 (Observation)

将检索结果反馈给 LLM，评估文档与问题的相关性是否足够

context = retrieved_docs
relevance = evaluate()
// STEP 04

✅ 判断结束

信息足够则生成最终回答，否则继续循环检索直到达到最大次数

if sufficient: respond()
else: loop continue
🔄 循环状态: 迭代中
📊 检索次数: 动态
✅ 最大迭代: 5次
