Agentic RAG 核心循环

Thought → Action → Observation → 迭代评估

🧠 LLM 决策 Think
🔍 检索工具 Action
📊 评估结果 Observe
❓ 判断结束 Done?
🔄 Agent
♻️ 循环迭代直到获得足够信息

🧠 LLM 决策 (Thought)

Agent 分析用户问题，决定是否需要检索，以及使用什么工具

model.bind_tools([rag_tool])
agent.prompt

🔍 执行检索 (Action)

调用 RAG 工具执行向量检索，获取相关文档片段

retriever.get_relevant_documents(query)

📊 观察评估 (Observation)

将检索结果反馈给 LLM，评估信息是否足够回答问题

context + retrieved_docs

✅ 判断结束

如果信息足够则生成回答，否则继续循环检索

if sufficient: 生成回答
else: 继续检索
