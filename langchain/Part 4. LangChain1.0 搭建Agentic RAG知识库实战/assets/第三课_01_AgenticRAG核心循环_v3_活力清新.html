Agentic RAG 核心循环

智能迭代问答流程

第一步 🧠 LLM 决策
→
第二步 🔍 执行检索
→
第三步 📊 评估结果
→
第四步 ❓ 判断结束
🔄 循环判断
✅ 信息足够 → 生成回答
↩️ 信息不足 → 继续检索
🧠

LLM 决策 (Thought)

Agent 分析用户问题，决定是否需要检索以及使用什么工具

智能决策
🔍

执行检索 (Action)

调用 RAG 工具从向量数据库获取相关文档片段

工具调用
📊

观察评估 (Observation)

将检索结果反馈给 LLM，评估信息是否足够回答问题

结果评估
✅

判断结束

如果信息足够则生成回答，否则继续循环检索

迭代控制
🔄 循环迭代直到获得足够信息或达到最大迭代次数（通常 3-5 次）
