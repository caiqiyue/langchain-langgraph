## <center>第一阶段、项目概述</center>

### 1. 系统架构

DeepAgents 爬虫自动化系统基于 LangGraph 构建，包含三大核心组件：

![爬虫Agent系统架构](../assets/第一课_01_爬虫架构.html)

**图解说明**：Agent Core（LangGraph编排）→ Tools（4个子Agent）→ Sandbox（Docker隔离环境）

### 2. 核心模块

| 模块 | 职责 |
|------|------|
| agent_config | 全局配置中心 |
| spider_demo | 双模式机制（标准/快速） |
| agent | 基于LangGraph的智能体编排 |
| docker_backend | Docker容器生命周期管理 |
| sandbox | 沙箱环境工厂 |
| tools | 工具定义（@tool装饰器） |

### 3. 双模式机制

| 模式 | 适用场景 |
|------|---------|
| 标准智能体模式 | 复杂、需要多次迭代 |
| 快速管道模式 | 简单、常见场景 |

### 4. 8步执行流程

| 步骤 | 内容 |
|------|------|
| 1 | 环境启动（Docker 容器初始化） |
| 2 | 任务接收（用户输入 URL/需求） |
| 3 | 网页获取（fetch_url 保存 HTML） |
| 4 | 结构分析（WebAnalyzer 分析 DOM） |
| 5 | 代码生成（CodeGenerator 生成爬虫） |
| 6 | 沙箱执行（DebugAgent 在 Docker 中运行） |
| 7 | 数据处理（DataProcessor 清洗去重） |
| 8 | 结果反馈（返回抓取的数据） |
