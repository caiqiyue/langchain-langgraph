LangGraph 框架基础入门

🎯 核心设计思想

State

共享的数据结构，在图的各个节点之间传递信息

Nodes

可执行的功能单元，接收State并返回更新的State

Edges

定义节点间的流转逻辑，控制工作流的执行路径

✨ 核心优势
  • ✓基于LangChain表达式语言(LCEL)构建
  • ✓支持循环图结构，突破线性序列局限
  • ✓内置状态管理，自动维护和传递状态
  • ✓高度灵活性，可在Python函数中定义任意逻辑
  • ✓良好的扩展性，轻松添加新节点和边
  • ✓支持多种大模型接入
