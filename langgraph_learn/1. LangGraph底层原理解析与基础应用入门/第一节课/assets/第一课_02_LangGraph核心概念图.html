LangGraph 四大核心概念

1 Nodes 节点

节点是 Python 函数，接收当前 State 作为输入，执行计算后返回更新的 State。每个节点代表一个具体的操作单元。

2 Edges 边

边定义节点间的连接关系，决定逻辑如何路由。包括普通边、条件边、入口点和结束点。

3 State 状态

状态是在图执行过程中维护和更新的共享数据。每个节点可以读取和写入状态通道，实现节点间数据传递。

4 Router 路由

路由使用 if-else 结构决定下一个节点。可以通过提示工程、输出解析器或工具调用实现。

StateGraph = Nodes + Edges + State + Router

这四个概念共同构成了 LangGraph 的核心架构，用于构建复杂的有状态、多参与者应用程序
