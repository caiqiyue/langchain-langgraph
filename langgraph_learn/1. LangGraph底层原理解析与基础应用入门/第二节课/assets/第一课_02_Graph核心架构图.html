编译后的可执行图

🔧 Graph 核心属性

  • nodes存储图中所有节点
  • edges存储图中所有边
  • branches存储条件分支
  • support_multiple_edges是否支持同一对节点间的多条边
  • compiled图是否已被编译

⚡ 核心方法

  • add_node()添加节点
  • add_edge()添加边
  • add_conditional_edges()添加条件边
  • set_entry_point()设置入口点
  • set_finish_point()设置结束点
  • compile()编译图

🏗️ StateGraph 特性

  • state_schema定义图状态的结构
  • config_schema定义配置的结构
  • 继承关系继承自Graph类
  • 状态更新通过Reducer函数更新状态

📝 代码示例

from langgraph.graph import StateGraph

# 定义状态
class State(TypedDict):
    messages: list

# 创建图
graph = StateGraph(State)
graph.add_node("chat", chat_node)
graph.add_edge("start", "chat")
graph.add_edge("chat", "end")

# 编译并执行
app = graph.compile()
result = app.invoke({"messages": []})
