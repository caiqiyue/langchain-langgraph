轨迹，项目在不同条件下的执行路径记录

Runs

运行，特定轨迹的单次执行记录

⚙️环境配置

# 设置环境变量
import os

os.environ["LANGCHAIN_API_KEY"] = "your-key"
os.environ["LANGCHAIN_TRACING_V2"] = "true"
os.environ["LANGCHAIN_PROJECT"] = "my-project"
  • 创建 LangSmith 账户
  • 获取 API Key
  • 配置环境变量
