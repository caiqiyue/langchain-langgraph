State 状态模式定义

📝字典类型定义状态

# 方式一：使用 dict
graph = StateGraph(dict)

graph.invoke({"key": "value"})
  • 简单直接，无需定义结构
  • 灵活性高，键值可动态添加
  • 缺少类型提示和验证

🏷️TypedDict 类型定义

# 方式二：使用 TypedDict
from typing import TypedDict

class State(TypedDict):
    messages: list
    count: int

graph = StateGraph(State)
  • 提供类型提示和检查
  • IDE 支持更好
  • 推荐用于生产环境
