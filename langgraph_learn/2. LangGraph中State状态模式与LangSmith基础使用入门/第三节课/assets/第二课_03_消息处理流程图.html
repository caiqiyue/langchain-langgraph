大模型处理

接入 LLM 处理对话

👤HumanMessage

from langchain_core.messages
  import HumanMessage

HumanMessage(
    content="用户输入内容"
)
  • •表示用户输入消息
  • •存储在 messages 列表
  • •触发 AI 响应

🤖AIMessage

from langchain_core.messages
  import AIMessage

AIMessage(content="AI响应")
  • •表示大模型响应
  • •由 LLM 节点生成
  • •追加到消息历史

💾operator.add

from typing import Annotated
import operator

class State(TypedDict):
    messages: Annotated[
        list, operator.add
    ]
  • •消息追加而非覆盖
  • •保留完整对话历史
  • •适合多轮对话场景
HumanMessage: 你好
→
AIMessage: 你好！有什么可以帮助你的？
→
AIMessage: 用户问了一个问题...

消息处理是构建对话机器人的基础，通过 operator.add 实现消息历史累积
