事件流的应用

📊

values

流式输出完整状态值

🔄

updates

流式输出增量更新

🐛

debug

输出调试详细信息

💬

messages

输出消息事件流

🔧流模式对比

模式 输出内容 适用场景
values 完整状态快照 需要最终结果
updates 节点增量更新 实时进度展示
debug 完整调试信息 问题排查
messages 消息事件 对话应用

⚡事件机制

  • async/await 异步流处理
  • 实时获取执行状态
  • 支持事件过滤
  • 可中断和恢复

📝代码示例

# 流式调用
async for event in app.astream(
    {"messages": [("user", query)]},
    stream_mode="updates"
):
    print(event)
