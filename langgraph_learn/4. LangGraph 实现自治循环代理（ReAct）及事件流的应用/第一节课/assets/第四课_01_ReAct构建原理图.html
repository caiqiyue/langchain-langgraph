ReAct vs Tool Calling

特性 ReAct Agent Tool Calling Agent
决策方式 Reasoning + Action 交替 Function Calling 直接
循环支持 内置循环机制 通常单轮
灵活性 高（可自定义循环） 中（结构固定）
适用场景 复杂推理任务 结构化工具调用

核心概念

  • Reasoning：推理思考，理解当前状态
  • Action：执行动作，调用工具
  • Observation：观察结果，处理响应
  • Loop：循环控制，决定是否继续

代码示例

from langgraph.prebuilt
  import create_react_agent

agent = create_react_agent(
    model,
    tools=[search, weather]
)

result = agent.invoke({
    "messages": [("user", query)]
})
