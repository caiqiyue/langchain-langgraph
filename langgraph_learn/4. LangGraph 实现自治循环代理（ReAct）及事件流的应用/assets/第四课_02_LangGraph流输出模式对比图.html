LangGraph 流输出模式对比图

values 状态完整值
说明：在图中的每个步骤之后流式传输状态的完整值
适用场景：
- 需要查看完整状态
- 调试状态变化
- 获取最终结果
updates 状态增量
说明：在图中的每个步骤之后将更新流式传输到状态
适用场景：
- 只需增量变化
- 减少数据传输
- 实时监控进度
debug 调试信息
说明：在整个图的执行过程中流式传输尽可能多的信息
适用场景：
- 深度调试
- 性能分析
- 问题排查
messages Token级别
说明：记录每个messages中的增量token
适用场景：
- 流式显示AI输出
- Token消耗监控
- 实时对话体验
