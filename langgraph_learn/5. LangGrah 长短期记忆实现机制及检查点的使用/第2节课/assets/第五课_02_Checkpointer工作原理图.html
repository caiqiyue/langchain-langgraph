Checkpointer 工作原理

1
创建 Checkpointer
在编译图时传入checkpointer参数，使图具备状态保存能力
memory = MemorySaver() graph = builder.compile(checkpointer=memory)
2
状态自动保存
每个super-step完成后，checkpointer自动存储中间状态信息
3
线程隔离
通过thread_id标识不同对话线程，实现状态隔离
config = {"configurable": {"thread_id": "1"}}
4
状态恢复
恢复对话时，通过相同thread_id获取历史状态

MemorySaver

内存存储，适用于开发和测试环境

SqliteSaver

持久化存储，适用于生产环境
