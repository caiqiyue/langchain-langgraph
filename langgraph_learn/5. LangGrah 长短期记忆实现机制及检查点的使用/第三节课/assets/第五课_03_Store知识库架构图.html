Store 知识库架构

命名空间 (Namespace)

用于对存储数据进行逻辑分组的标识符

namespace = (user_id, "memories")

存储操作

支持 put、get、search、delete 等操作

store.put(namespace, id, data) store.get(namespace, id) store.search(namespace)

跨线程共享

Store中的数据可在不同线程间共享访问

config = {"configurable": {"thread_id": "10"}}

Store 与 Checkpointer 的区别

Checkpointer
线程内状态保存
Store
跨线程信息共享
