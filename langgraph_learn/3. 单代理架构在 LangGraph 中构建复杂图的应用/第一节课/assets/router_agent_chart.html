Router Agent 条件路由结构

条件边 flow

    START
       │
       ▼
    ┌─────────────────┐
    │  chat_with_model │  (结构化输出)
    └────────┬────────┘
             │
             ▼
    ┌─────────────────┐
    │ Router Function │  (isinstance 判断)
    └────────┬────────┘
             │
       ┌─────┴─────┐
       ▼           ▼
   ┌────────┐  ┌────────┐
   │insert_db│  │final_  │
   │ (UserInfo)│ │answer  │
   └────┬────┘  │(Conv..)│
        │        └───┬───┘
        ▼            │
       END            ▼
                   END

说明

  • chat_with_model: 使用 with_structured_output 生成结构化输出
  • Router Function: 通过 isinstance 判断输出类型，决定路由分支
  • insert_db: 当输出为 UserInfo 时，执行数据库插入
  • final_answer: 当输出为 ConversationalResponse 时，直接生成响应
