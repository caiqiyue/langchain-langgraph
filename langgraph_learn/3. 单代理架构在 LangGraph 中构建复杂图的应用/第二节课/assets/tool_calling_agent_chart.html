工具调用 flow

    用户输入
       │
       ▼
    ┌─────────────────┐
    │ chat_with_model  │  (LLM + bind_tools)
    └────────┬────────┘
             │
             ▼
    ┌─────────────────┐
    │  tool_calls?    │  (exists_function_calling)
    └────────┬────────┘
             │
       ┌─────┴─────┐
       ▼           ▼

       │           │

    ┌──────────┐  ┌──────────┐

         └─────┬──────┘
               │
               ▼
             END

ToolNode 工作原理

    tools_by_name = {tool.name: tool for tool in tools}

    def tool_node(state: dict):
        result = []
        for tool_call in state["messages"][-1].tool_calls:
            tool = tools_by_name[tool_call["name"]]
            result.append(tool.invoke(tool_call["args"]))
        return {"messages": result}

说明

  • chat_with_model: 大模型根据输入决定是否调用工具
  • exists_function_calling: 判断是否存在 tool_calls
  • execute_function: 执行 ToolNode 进行工具调用
  • natural_response: 结合工具结果生成最终响应
