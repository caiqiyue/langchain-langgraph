Breakpoint 断点工作原理

compile() 方法核心参数

checkpointer 接收任意类型的checkpointer，用于保存图的状态
interrupt_before 在指定节点之前中断图的运行
interrupt_after 在指定节点之后中断图的运行
# 编译图时添加断点 memory = MemorySaver() graph = builder.compile( checkpointer=memory, interrupt_before=["execute_users"] ) # 查看当前状态 snapshot = graph.get_state(config) # 更新状态（人工审批） graph.update_state(config, {"user_approval": "是"}) # 继续执行 async for chunk in graph.astream(None, config): print(chunk)
